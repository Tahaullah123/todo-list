# todo-list

Dette er en veldig enkel todo-liste laget med React. Du kan skrive inn oppgaver, se dem i en liste, markere dem som ferdige og slette dem. Prosjektet var mest for å lære litt om hvordan React fungerer.

## Hvorfor jeg laget prosjektet

Jeg kan ikke så mye React, så dette prosjektet var en måte å prøve det ut på en enkel måte. Jeg ville forstå hvordan man lager komponenter, hvordan man lagrer data i state, og hvordan man oppdaterer ting på skjermen.

## Hva appen kan

- Legge til en oppgave  
- Markere en oppgave som ferdig  
- Slette en oppgave  
- Vise alt i en enkel liste

## Hva jeg brukte

- React  
- JavaScript  
- Litt CSS for enkel styling
